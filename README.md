## 4軸の意味

| 軸 | コード | A側 | B側 |
|----|--------|-----|-----|
| 動力源 (Q1–10) | E / I | E: 影響型（人を動かすことで燃える） | I: 設計型（構造を作ることで燃える） |
| 判断軸 (Q11–20) | L / S | L: 論理型（データ・数字で判断） | S: 感性型（直感・ストーリーで判断） |
| 行動スタイル (Q21–30) | P / A | P: 戦略型（設計してから動く） | A: 実行型（動きながら修正する） |
| 目的志向 (Q31–40) | F / C | F: 自律型（自分の自由のために動く） | C: 貢献型（誰かのために動く） |

## 使い方

ブラウザで `index.html` を直接開くか、GitHub Pages で公開してください。

### GitHub Pages での公開

1. このリポジトリを GitHub にプッシュ
2. Settings > Pages > Source: Deploy from a branch
3. Branch: `main` / `(root)` を選択して Save
4. 数分後に `https://<username>.github.io/<repo-name>/` で公開されます

## ファイル構成

```
for-life-shindan/
├── index.html   # メインHTML（5画面：スタート・質問・結果・タイプ一覧・軸解説）
├── style.css    # スタイル（CSS変数 + モバイルファースト）
├── script.js    # 診断ロジック・16タイプデータ（Vanilla JS）
└── README.md    # このファイル
```

- アニメーション: CSS トランジション・キーフレームのみ
