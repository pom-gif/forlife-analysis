## 4軸の意味

| 軸 | A側 | B側 |
|----|-----|-----|
| 動力源 (Q1–6) | E: 影響型（人を動かすことで燃える） | I: 設計型（構造を作ることで燃える） |
| 判断軸 (Q7–12) | L: 論理型（データ・数字で判断） | S: 感性型（直感・ストーリーで判断） |
| 行動スタイル (Q13–18) | P: 戦略型（設計してから動く） | A: 実行型（動きながら修正する） |
| 目的志向 (Q19–24) | F: 自律型（自分の自由のために動く） | C: 貢献型（誰かのために動く） |

## 使い方

ブラウザで `index.html` を直接開くか、GitHub Pages で公開してください。

### GitHub Pages での公開

1. このリポジトリを GitHub にプッシュ
2. Settings > Pages > Source: Deploy from a branch
3. Branch: `main` / `(root)` を選択して Save
4. 数分後に `https://<username>.github.io/<repo-name>/` で公開されます

## ファイル構成

```
for-life-shindan/
├── index.html   # メインHTML（3画面を1ファイルで管理）
├── style.css    # スタイル（CSS変数 + モバイルファースト）
├── script.js    # 診断ロジック（Vanilla JS）
└── README.md    # このファイル
```

## 技術仕様

- 外部ライブラリなし（Vanilla JS のみ）

- アニメーション: CSS トランジション・キーフレームのみ
- モバイルファースト、max-width: 720px センタリング
